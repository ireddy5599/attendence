Attendance Setup

Your attendance is stored as a JSON file in a private GitHub Gist — no server, no database needed. Data lives in your GitHub account.

You need a GitHub token with only the gist scope. It's saved only in your browser, never sent anywhere else.

GitHub Personal Access Token
  1. Go to github.com/settings/tokens
  2. Click Generate new token (classic)
  3. Set note, check only the gist scope
  4. Generate → copy → paste above
